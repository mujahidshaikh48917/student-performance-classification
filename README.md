# Student Performance Classification

## 📌 Project Overview
This project uses Machine Learning to classify student performance into High, Average, and Low categories.

## 📊 Dataset
The dataset contains student academic records including grades, study time, failures, and attendance.

## ⚙️ Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## 🤖 Model Used
Decision Tree Classifier

## 📈 Results
The model achieved good accuracy in predicting student performance categories.

## 🎯 Conclusion
This system can help educators identify weak students and take early action.
